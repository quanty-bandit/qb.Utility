# qb.Utility
Utility glob

# CONTENT

**LocalizationUtility**

Provides utility methods for converting between SystemLanguage enums, CultureInfo codes, and English language names for localization purposes. 

**WebRequestUtility**

Provides utility methods for performing web requests, handling JSON deserialization, and checking file existence over HTTP.

**ZipUtility**

Provides utility methods for compressing and decompressing strings using GZip.
